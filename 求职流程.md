# 一、明确诉求

办公地点 + 薪资待遇 + 工作内容



# 二、搜集公告

各种招聘APP（Boss、猎聘、智联）



# 三、解读 JD

**对方给什么：**

-  福利待遇：工资结构（底薪、绩效、年终奖）、五险一金、休假等
-  内部情况：公司具体信息太少、JD写的不好、HR态度与能力等



**对方要什么：**

-  具体职责：解读核心任务，在你简历和面试中体现拥有这方面的能力。
   -  核心动词**：**如“负责”、“主导”、“管理”、“协调”、“协助”、“分析”、“优化”等动词，说明了主动权和责任范围。

-  工作要求：从事这份工作，必须掌握的技能与经验。

   -  隐性要求：“具备抗压能力” -> 可能经常加班。“能在快节奏环境下工作” -> 可能经常有紧急任务，需要加班。

      

# 四、定制简历 & 面试准备

先根据JD要求（专业技能、工作经历、项目经验、个人方面）定制简历（可以不会），然后按简历所写以面试标准进行准备。



# 五、投递沟通

看着满意的，只要自身差距无法逾越就投。
